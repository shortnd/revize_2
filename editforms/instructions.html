Revize Edit Forms Instructions

Intro

Going forward every HTML that gets sent to be enabled will have a folder called editforms and in this form you will see all the editforms need for any size site

Each form refers to a “plugins/_editforms_” folder at the the root of the server there we have all the scripts plugins and includes needed for each form

Please never modify the include files at the root of the server

If you need an edit form that is not in the list or please ask Derek or Samir to create it and one of them will give you the html

If you are not sure of something please ask Derek or Samir

Extra fields have been added to all forms. Anything not need just comment it out

Never have edit forms open in popup windows please

Scripts

We have included the setup script in the include files so there is no need to add them

In head:

Editform: 
<%@ include file="/util/setup_editform_before_endhead.jsp" %>
Editlist: 
<%@ include file="/util/setup_editlist_before_endhead.jsp" %>

Start Body (before form):

Editform: 
<%@ include file="/util/setup_editform_javascript.jsp" %>
Editlist: 
<%@ include file="/util/setup_editlist_javascript.jsp" %>

End Body (after form) :

Editform: 
<%@ include file="/util/setup_editform_before_endbody.jsp" %>
Editlist: 
<%@ include file="/util/setup_editlist_before_endbody.jsp" %>
